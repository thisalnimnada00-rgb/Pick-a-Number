SOLO MODE
Pick any category → choose a question → answer honestly. Pure fun!
MULTIPLAYER MODE 🔥
• Create or join a room with a friend
• One player picks a question for the OTHER player
• The question instantly appears on their screen
• They type an answer → everyone sees it
• Take turns roasting / learning about each other
